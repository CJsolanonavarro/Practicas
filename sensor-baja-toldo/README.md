# Sensor baja toldo

Construcción de un sensor para bajar el toldo de una estancia.

![](practica.gif)

## Materiales

- 1 Placa de Arduino
- 1 Protoboard
- 8 Latiguillos (macho-macho)
- 1 Leds
- 2 Resistencias
- 1 Sensor LDR
- 1 Servomotor
